This is my form page, and I am going to explain how it works. When we first got the code for the form, it had unclickable buttons and did not have boxes that changed color when clicked and had little style. The first thing we did was style the page by changing the headers, the colors, and text of the page. Then, we made some webkits so the page would keep its style even if the web browser could not handle the code. Next we got rid of the code that made the checkboxes. Then, we used an svg image of a check box and put that in place of the other boxes and made it clickable. Next we added inputs for the email and styled the buttons so they changed color when hovered over and also added an svg image for an email icon. We also made the textboxes change color slighty when focused, which basically means that they change color when clicked and stay that way untill clicked off of.
